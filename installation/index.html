Toolchain

The toolchain consist of three major components:

  • IceStorm Tools - Bitstream format documentation and tools
  • Arachne-PNR - The place and route tool
  • Yosys - Verilog synthesis tool
  • icoprog - icoBoard programming tool (needs to be run on the RPi)

Installation of the tools is described in detail on the icestorm website under the section Where are the Tools? How to install?.

Raspberry Pi (RPi)

If you bought the icoBoard bundle you have everything preinstalled on the SD card to get you started - just hook up your input devices of choice and your HDMI screen, or simply connect via SSH.
